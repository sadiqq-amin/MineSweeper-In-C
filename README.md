# Minesweeper in C

A console-based implementation of the classic **Minesweeper** game written entirely in **C**. This project was developed as a first-semester university group project for the **CS1002 â€“ Programming Fundamentals** course, applying core programming concepts learned during the semester.

---

## Features

- Console-based Minesweeper gameplay
- Grid-based board with hidden mines
- User interaction through keyboard input
- Real-time board updates
- Win and lose conditions implemented
- Written completely in C

---

## Technologies Used

- C Programming Language
- Standard C Libraries
- Console / Terminal Interface

---

## Project Purpose

This project was created to strengthen understanding of fundamental programming concepts, including:
- Conditional statements
- Loops
- Functions
- Arrays
- Basic game logic
- Input/output handling in C

---

## How to Run

1. Download or clone the repository.
2. Make sure all project files are in the same directory.
3. Compile the program using a C compiler (e.g., GCC):

```bash
gcc minesweeper.c -o minesweeper
```

4. Run the game:

```bash
./minesweeper
```

---

## Usage Instructions

- Follow on-screen instructions to play the game.
- Enter coordinates to uncover cells.
- Avoid mines to win the game.
